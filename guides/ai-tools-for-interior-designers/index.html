Automation Workflows for Designers

New Inquiry Response Sequence

  1. Instant acknowledgment with consultation booking link
  2. Pre-consultation questionnaire to qualify and prepare
  3. Reminder with portfolio highlights before meeting
  4. Post-consultation thank you and next steps
  5. If no decision in 5 days, follow up with project timeline

Proposal Follow-Up Sequence

  1. Proposal delivery with personalized note
  2. Day 3: Check-in asking if questions arose
  3. Day 7: Share similar project example or testimonial
  4. Day 14: Gentle reminder about timeline and availability
  5. Day 21: Final follow-up offering to modify scope

Project Milestone Communication

  1. Project kickoff welcome and timeline overview
  2. Weekly progress updates with visual previews
  3. Delivery milestone celebrations
  4. Installation scheduling and preparation
  5. Project completion with professional photography
