Automation Workflows for Yoga Studios

New Student Journey

  1. Welcome email with studio guidelines and what to bring
  2. Post-first-class check-in and feedback request
  3. Class suggestions based on experience level
  4. Mid-intro-package check-in on progress
  5. Membership conversion offer before package expires

Member Engagement Sequence

  1. Weekly class recommendations based on attendance patterns
  2. If no classes in 2 weeks: gentle check-in message
  3. If no classes in 4 weeks: "We miss you" with motivation
  4. Workshop and special event invitations
  5. Monthly studio newsletter with community updates

Membership Renewal Automation

  1. 30-day notice before renewal with usage summary
  2. Highlight benefits used and progress made
  3. 14-day reminder with renewal options
  4. 7-day final reminder
  5. Post-lapse reactivation sequence if needed
